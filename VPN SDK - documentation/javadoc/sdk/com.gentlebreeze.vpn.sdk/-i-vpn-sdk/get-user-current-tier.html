getUserCurrentTier

abstract fun getUserCurrentTier(): Single<Tier>

Get an observable of the user current tier.

This observable will return a Tier subclass.

Tier is a kotlin sealed class with three children:

  • SubscriptionTier
  • BandwidthTier
  • EmptyTierTier

This Single will return any of these three children.

If user has a tier of any kind a SubscriptionTier or BandwidthTier will be emitted.

If the user doesn't have any current tier, a EmptyTier will be emited

On Failure, observable will return one of the following throwables through onError:

  • ApiCallFailure
  • NetworkNotAvailableFailure

Return
a single observable with the user's tier
